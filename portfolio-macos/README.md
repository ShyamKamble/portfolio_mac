## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/username/macos-portfolio.git

# Navigate to project directory
cd macos-portfolio

# Install dependencies
npm install

# Start development server

```

### Build for Production

```bash
# Create production build
npm run build

# Start production server
npm start
```
